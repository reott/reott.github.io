3D Cave Generation

Ziel: Generierung einer Höhle mithilfe von Algorithmen

Technologien: Unity, C#, Shader Graph

Zeit: 1 Woche

Details: Zur Generierung des Mesh wurde Perlin Noise in Verbindung mit Marching Cubes benutzt. Im Bereich Texturen konnte ich Erfahrungen mit UV Mapping (per Code) und Triplanar Mapping (per Shader Graph) sammeln. Ein simpler Spawn-Algorithmus verteilt Kristalle, Pflanzen, etc. gleichmäßig nach bestimmten Regeln in der Höhle. Einen kleinen Einblick in das Particle System von Unity konnte ich hier auch gewinnen.

Forschungsprojekt Nutzung von KI zur Bearbeitung von Anträgen

Ziel: Erarbeiten wie KI zur Unterstützung von Sachbearbeitern beim Bearbeiten von Anträgen eingesetzt werden kann

Technologien: Python, LangChain, ChatGPT API

Zeit: mehrere Wochen

Details: Zusammen mit einem kleinen Team haben wir einen Prozess unseres Kunden analysiert und Ideen erarbeitet, wie KI hier unterstützend eingesetzt werden kann. Als Ergebnis dieses Projekts haben wir ein kleines Tool entwickelt, welches Informationen aus Anträgen mit Retrieval Augmented Generation ziehen kann und diese in eine Vorlage zur Weiterverarbeitung einfügt.
